steamcard-dl
===============================

A python script which downloads all the steam card wallpapers and backgrounds via http://www.steam.cards/

Requirements
===============================

- Python 2
- [BeautifulSoup4](http://www.crummy.com/software/BeautifulSoup/)

Usage
===
```sh
python steamcard-dl.py
```
