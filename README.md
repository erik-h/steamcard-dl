Steam-Card-Wallpaper-Downloader
===============================

This python script downloads all the steam card wallpapers via http://www.steam.cards/

Requirements
===============================

[BeautifulSoup4](http://www.crummy.com/software/BeautifulSoup/) - To parse the html
